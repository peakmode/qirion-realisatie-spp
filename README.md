# Visie Directeur Realisatie — Qirion SPP-rapport

Strategisch visiedocument voor de Directeur Realisatie van Qirion: hoe ziet de strategische personeelsplanning eruit, hoe ontwikkelt de markt zich, en wat betekent dat voor de afdeling Realisatie (~742 medewerkers).

**Live:** https://peakmode.github.io/qirion-realisatie-spp/

## Inhoud

- **Tab 01 — De Opgave:** macro-transitie, TenneT €50 mrd pipeline, Engineering-Realisatie scheefgroei, AI-heatmap, dubbele druk
- **Tab 02 — Strategische Personeelsplanning:** rollenlandschap, arbeidsmarkt, aannemersmarkt, scenario's 2030
- **Tab 03 — Bouwstenen & Routekaart:** strategische keuzes, technologische versnellers, 90-dagen plan, versterkings-vragen
